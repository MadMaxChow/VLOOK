若没有指定题注，VLOOK™ 会自动以表格文字内容（头尾片断）作为题注

[+] 设置方式一

  • 在表格前添加 1～6 级 任间一级标题

  • 对应的快捷键操作：

    • Windows：⌃ Ctrl ＋ 1～6
    • macOS： ⌘ Command ＋ 1～6

  • 对应的 Markdown 格式语法为：###### 这是题注内容

  • 同时这种方式下会自动启用表格的逐行自动编号特性。

[+] 设置方式二

  • 将作为题注内容的整段文字设置为指定的格式：斜体＋高亮

  • 对应的快捷链操作：

    • Windows：⌃ Ctrl ＋ i ，松开后 ⌃ Ctrl ＋ ⇧ Shift ＋ H
    • macOS： ⌘ Command ＋ i ，松开后 ⌘ Command ＋ ⇧ Shift ＋ H

  • 对应的 Markdown 格式语法为：*==这是题注内容==* 。

双题注

VLOOK™ 支持双题注，第二个题注显示在表格下方，同时还支持在 Typora 编辑时预览（在导出 HTML 后正式生效）。

设置方式：

  • 第一步：先参考以上的单题注的设置方式二完成；
  • 第二步：在以上题注内容的最后添加第二个题注的内容，并用下划线包裹；
  • 对应的 Markdown 格式语法为：*==这是第一个题注内容_这是第二个题注_==* 。
